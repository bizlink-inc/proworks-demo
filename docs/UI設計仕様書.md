## レイアウト

### テンプレート3種

#### ① センター寄せ台紙（ログイン・新規登録）
- **レイアウト形式**: センター寄せ（両側に余白あり）
- **用途**: ログイン、新規登録などの認証ページ

#### ② 左サイド サブメニューあり（マイページ）
- **レイアウト形式**: 左にサブメニュー、右にコンテンツ
- **用途**: マイページなど

#### ③ サブメニューなし（案件一覧・応募済み案件）
- **レイアウト形式**: フルコンテンツ表示
- **用途**: 案件一覧、応募済み案件ページ

### レスポンシブ対応

#### Tablet
- レイアウトの縮小版
- 左サイドメニューはそのまま表示

#### Mobile
- ハンバーガーメニュー導入
- レイアウトを横幅に適応
