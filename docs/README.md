## 💡 ドキュメント作成時のルール

1. **命名規則**: 日本語で簡潔に。階層がある場合は `テーマ_詳細.md`
2. **テンプレート**: 各フォルダーのREADMEに記載
3. **定期更新**: 実装完了時に最新情報に更新
4. **参照関係**: 他のドキュメントを参照する場合は相対パスで記載

例: `[基本設計](../01_基本設計/ProWorks基本設計.md)`

## 🔍 ドキュメント検索のコツ

- **機能実装について**: `03_機能実装/` を参照
- **インフラ・コスト**: `02_インフラ・アーキテクチャ/` を参照
