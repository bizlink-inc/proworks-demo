## 1. レイアウト（テンプレート 3 種）

### ① センター寄せ台紙

- **用途**: ログイン・新規登録
- **特徴**: コンテンツを中央に配置したシンプルなレイアウト

### ② 左サイド サブメニューあり

- **用途**: マイページ
- **特徴**: 左側にサブナビゲーション、右側にコンテンツエリア

### ③ サブメニューなし

- **用途**: 案件一覧・応募済み案件
- **特徴**: 全幅でコンテンツを表示

### レスポンシブ対応

| デバイス | 特徴                       |
| -------- | -------------------------- |
| Desktop  | フルレイアウト表示         |
| Tablet   | サイドメニューがドロワー化 |
| Mobile   | ハンバーガーメニュー化     |

## CSS 変数（参考実装）

```css
:root {
  /* 背景色 */
  --bg-body: #f3f9fd;
  --bg-card: #ffffff;
  --bg-input-focus: #e8f0fd;
  --bg-alert: #fecfd7;
  --bg-alert-light: #ffe3e8;
  --bg-disabled: #f2f2f2;

  /* 枠線・ライン */
  --border-primary: #d5e5f0;
  --border-secondary: #9ab6ca;
  --border-input: #c3c1c1;
  --border-divider: #999999;

  /* ボタン・リンク */
  --color-primary: #63b2cd;
  --color-primary-dark: #1f3151;
  --color-secondary: #686868;
  --color-link: #63b2cd;
  --color-link-alt: #3966a2;

  /* テキスト */
  --text-heading: #30373f;
  --text-body: #000000;
  --text-sub: #686868;
  --text-muted: #999999;

  /* アラート・サポート */
  --color-success: #3f9c78;
  --color-warning: #fa8212;
  --color-error: #d22852;
  --color-price: #ea8737;

  /* シャドウ */
  --shadow-card: rgba(188, 210, 225, 0.28);

  /* 角丸 */
  --radius-sm: 4px;
  --radius-md: 12px;

  /* フォント */
  --font-family: "Noto Sans JP", sans-serif;
}
```
