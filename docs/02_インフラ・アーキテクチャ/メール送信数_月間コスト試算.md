# Part 2: Amazon SES 料金

## Amazon SES 料金（公式情報）

| 項目       | 料金                                         |
| ---------- | -------------------------------------------- |
| 送信メール | **$0.10 / 1,000 通**（= 1 通あたり $0.0001） |
| 無料枠     | 3,000 メッセージ/月 × 最初の 12 ヶ月         |

※ 為替レート: $1 = ¥150 で換算  
※ 1 通あたり約 **¥0.015**（1,000 通あたり ¥15）  
※ 長期運用を想定し、**無料枠は計算に含めない**

---

## 月間メール送信コスト（Amazon SES）

| 累計ユーザー | 月間通数  | Amazon SES 月額コスト |
| ------------ | --------- | --------------------- |
| 100 人       | 1,200 通  | **¥18**               |
| 500 人       | 6,000 通  | **¥90**               |
| 1,000 人     | 12,000 通 | **¥180**              |
| 3,000 人     | 36,000 通 | **¥540**              |
| 5,000 人     | 60,000 通 | **¥900**              |

✅ **この表は公式料金に基づいており、確度の高い情報です**

### 計算式

```
月額コスト（円）= 月間送信数 × $0.10 / 1,000 × ¥150
```

**例（1,000 人の場合）**:

```
12,000 通 × $0.10 / 1,000 × ¥150 = ¥180
```

# Part 4: コストの特徴

## Amazon SES のコストメリット

1. **完全従量課金**: 送信した分だけ課金されるため、無駄がない
2. **低コスト**: 1,000 通あたり約 ¥15 と非常に安価
3. **スケーラブル**: ユーザー数が増えても、送信数に比例してコストが増えるだけ
4. **予測可能**: 送信数が分かれば、コストを正確に予測できる

## コスト削減のポイント

- **無料枠の活用**: 最初の 12 ヶ月間は月 3,000 通まで無料
- **バッチ送信の最適化**: 不要なメールを減らすことでコスト削減
- **バウンス・苦情の管理**: 無効なアドレスへの送信を防ぐ
