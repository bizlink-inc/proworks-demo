# ミーティング議事録

このフォルダーには、会議の議事録、文字起こし、要約、検討結果が格納されています。

## 📂 フォルダー構成

各ミーティングごとにフォルダーまたはファイルを作成してください。

### 命名規則

- ファイル名: `YYYY-MM-DD_ミーティング名.md`
- フォルダー名: `YYYY-MM_テーマ` （複数回開催される場合）

例:
- `2025-12-05_キックオフミーティング.md`
- `2025-12_設計レビュー/` （複数回）

## 📝 ファイル形式

各ファイルには以下の構成を推奨します：

```
# ミーティング名

日時: YYYY-MM-DD HH:MM〜HH:MM
場所: オフィス / Zoom
参加者: 名前1, 名前2, ...

## 議題

- 議題1
- 議題2

## 決定事項

- 決定1
- 決定2

## アクションアイテム

| 担当者 | タスク | 期限 |
|------|-------|------|
| 名前1 | タスク1 | 2025-12-10 |

## 議論ポイント

- ポイント1: 詳細
- ポイント2: 詳細

## 次回予定

次回: YYYY-MM-DD HH:MM
```

## 🎯 対象者

- プロダクトマネージャー
- 技術責任者
- 開発チーム
- 関係者全員
